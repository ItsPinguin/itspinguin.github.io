Defining the sandwich
So what is a sandwich you may ask? Here we refer to it as 'stacking' functions. A n layers sandwich is a function which uses n functions.
Let's look at this beautiful sandwich equation:
\[ f(x) = \sin(3x + \tan( 2y + \sin(x))) \]
